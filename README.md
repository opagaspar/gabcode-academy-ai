# GabCode Academy AI

Plataforma premium de aprendizado de programação com IA, editor integrado e gamificação.

## Stack
- React 19
- TypeScript
- Vite
- TanStack Router
- Tailwind CSS v4
- Zustand
- Monaco Editor
- Framer Motion

## Scripts
```bash
npm install
npm run dev
npm run build
npm run preview
```

## Publicação na Vercel
Use estas configurações:
- Framework Preset: **Vite**
- Build Command: **npm run build**
- Output Directory: **dist**

O arquivo `vercel.json` já inclui rewrite para SPA.
